检查清单

  1. 打开浏览器开发者工具 (F12)
  2. 查看 Console 标签页中的错误信息
  3. 查看 Network 标签页检查资源加载情况
  4. 检查页面是否显示内容

常见问题

如果页面空白：

  • 检查浏览器控制台是否有 JavaScript 错误
  • 检查是否有 404 错误（CSS/JS 文件未找到）
  • 尝试硬刷新页面 (Ctrl+Shift+R 或 Cmd+Shift+R)
  • 检查是否有 CORS 错误
